# VitaTrace360 GitHub Pages Website

Static multipage presentation website for `vitatrace360.github.io`.

## Languages
- Romanian at root: `/index.html`
- English: `/en/`
- French: `/fr/`
- Spanish: `/es/`
- Italian: `/it/`
- German: `/de/`

## Pages
- Home
- Platform
- Modules
- Solutions
- Compliance
- Demo
- Contact
- Privacy
- Terms

## Deploy
1. Create a GitHub repository named `vitatrace360.github.io`.
2. Upload all files from this folder to the repository root.
3. Go to repository Settings → Pages.
4. Select branch `main` and folder `/root` if needed.
5. The site will be available at `https://vitatrace360.github.io`.

## Before public commercial use
- Replace `contact@example.com` in `assets/script.js` with your real email.
- Review Privacy Policy and Terms with a qualified legal advisor.
- Verify trademark/domain availability for VitaTrace360.
- Replace conceptual demo illustrations with real platform screenshots after MVP build.
